- Features
Feature	Description
- Live Price	Real-time bid/ask price updates
- Candlestick Chart	1H timeframe chart with price visualization
- AI Signal	BUY / SELL / WAIT signals generated by AI
- Technical Indicators	RSI(14), MACD, EMA 20/50/100/200
- Entry/SL/TP	Recommended entry, stop loss, and take profit levels
- Confidence Score	Signal reliability score (0–100%)
- Signal History	Full history log of all past signals
- Auto Refresh	Automatic analysis refresh at configurable intervals
- Multi Pair	Supports XAUUSD, EUR/USD, and other pairs
- Desktop App	Native Windows application via Electron
